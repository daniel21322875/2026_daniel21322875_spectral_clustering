
# Final Project: Spectral Clustering for Image Segmentation

This folder contains the final implementation and weekly development notes for the project **Spectral Clustering for Image Segmentation**.

The main goal of this project is to apply spectral clustering to image segmentation. Each pixel in an image is treated as a data point, and clustering is performed based on color features and spatial information.
