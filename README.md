
# Spectral Clustering for Image Segmentation

This repository contains the materials, testing scripts, weekly notes, and final implementation for the project **Spectral Clustering for Image Segmentation**.

The goal of this project is to apply spectral clustering to image segmentation. In this method, each pixel in an image is treated as a data point, and clustering is performed based on color features and spatial information.
